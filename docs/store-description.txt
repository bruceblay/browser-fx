Browser FX transforms your web audio experience with real-time professional audio effects. Capture any tab's audio and shape it with studio-style knobs in a sleek dark interface, complete with an audio-reactive cymatic visualizer that moves with your music.

22 HIGH-QUALITY EFFECTS:
Bitcrusher • CD Skipper • Reverb • Hall Reverb • Filter • Auto Filter • Comb Filter • Delay • Tap Delay • Distortion • Chorus • Phaser • Flanger • Tremolo • Vibrato • Auto Panner • Pitch Shifter • Ring Mod • Compressor • DJ EQ • Tape Stop • Lo-Fi Tape

KEY FEATURES:
• Real-time audio processing on any website
• Per-tab effects, run different effects on different tabs
• Studio-inspired interface with responsive knobs and per-effect color themes
• Audio-reactive cymatic visualizer, a Chladni plate pattern driven by your sound
• Live parameter adjustment while audio plays
• Full stereo processing
• Settings automatically saved per tab
• No account required, works instantly

PERFECT FOR:
Music streaming • Video content • Content creation • Audio experimentation • Lo-fi listening sessions

SIMPLE TO USE:
1. Click the Browser FX icon
2. Choose an effect
3. Press the power button
4. Twist the knobs in real time
5. Enjoy

PRIVACY FIRST:
All audio processing happens locally in your browser using the Web Audio API. No data collection, no external servers, complete privacy.
